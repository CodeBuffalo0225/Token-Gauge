# Token Gauge

Universal token usage tracker for Claude. Gas gauge for your context window.

## Three modes

| Mode | Command | Requires |
|------|---------|----------|
| Live API | `--mode live` | ANTHROPIC_API_KEY |
| Manual entry | `--mode manual` | Nothing |
| Estimate | `--mode estimate` | Nothing |

## Install

```bash
npm install
node index.js
```
